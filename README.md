AWAKE
=====

syon's front-end boilerplate.

- webpack
  - `webpack-dev-server`
    `source-map`
- Babel
  - `ES2015`
- PostCSS
  - `Autoprefixer`
    `PreCSS`
- Style
  - `Bootstrap`
    `jQuery`
    `Google Fonts`
    `FontAwesome`
- Template
  - `Pug (Jade)`


## start dev

```bash
$ npm start
```


## build

```bash
$ npm run build
```
